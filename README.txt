Использование:
1) В файле .env введите логин и пароль для входа в веб-конфигуратор
2) Выберите тип наклейки
3) При необходимости измените ориентацию на 90°
4) Две строки ввода над QR-кодом предназначены для технических наклеек.
	I) для названия
	II) для смещения по оси X
5) Если общее положение изображения сместилось, измените соответствующие координаты слева

Usage:
1) In the .env file, enter the login and password to log in to the Web Configurator
2) Select the sticker type
3) If necessary, change the orientation to 90°
4) The two input lines above the QR code are for technical stickers.
	I) for the name
	II) for the X-axis offset
5) If the overall image position has shifted, adjust the corresponding coordinates from the left
